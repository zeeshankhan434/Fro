# Frontend Deployment

This is the frontend for the Crypto Finance Blog.

## Deployment on Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory.
3. Set up the following environment variables in Vercel Dashboard:
   - `VITE_API_URL`: The URL of your deployed backend (e.g., `https://your-api.render.com`)

## Local Development
1. `npm install`
2. `npm run dev`
